# Discord 명령어 (한글)

이 문서는 운영에서 노출되는 공개 슬래시 커맨드와 UI 계약을 정리합니다.

## 공개 명령 표면
- `/홈 설치 [채널]`
- `/회의 시작|종료|기록|항목`
- `/안건 설정|오늘|최근`
- `/과제 목록|상세|등록|완료`
- `/모각코 오늘|내정보|랭킹`
- `/설정 마법사|상태`

## 시스템
- `/핑`
  - 설명: 봇 동작 확인
  - 응답: `pong` (ephemeral)

- `/도움말 [카테고리]`
  - 설명: 카테고리별 명령어 설명과 레거시 매핑 안내
  - 카테고리: `전체/회의/안건/과제/모각코/설정/홈`
  - 응답: 도움말 임베드(ephemeral)

## 홈 대시보드
- `/홈 설치 [채널]`
  - 설명: 홈 메시지를 보장 생성/복구하고 핀 상태를 점검
  - 권한: 운영진(미설정 초기에는 Manage Server/Administrator)

### HOME_V2 UI 계약
- 홈은 메뉴판이 아니라 상태판으로 동작
- 임베드 필드
  - `오늘 상태`: `안건 유무 · 진행 중 회의 유무 · 미완료 과제 수 · 오늘 모각코 참여 인원`
  - `지금 해야 할 일`: 가장 가까운 마감 1건 + 진행 중 회의 상태 + 안건 설정 상태
- 버튼/셀렉트
  - 상태별 1행 버튼
    - normal: `[회의 시작] [빠른 과제] [내 기록]`
    - meeting-active: `[진행 중 회의 열기] [빠른 과제] [내 기록]`
    - setup-incomplete: `[설정 시작] [내 기록] [도움말]`
  - 다음 행: `[더보기]` -> `안건 / 과제목록 / 모각코 / 설정 / 도움말`
  - 안건 존재 시: `[오늘 안건 링크]`
- `내 기록`은 ephemeral 응답으로 공개 채널 스팸을 방지

### 홈 채널 가드
- 홈 채널에서는 기능 실행 명령을 제한하고, 기능별 채널로 이동을 안내
- 전용 채널 미설정이면 `/설정 마법사` 실행을 안내

## 안건
- `/안건 설정 링크:<http/https> [제목]`
  - 설명: 오늘(Asia/Seoul) 안건 링크 등록/수정
  - 권한: 운영진

- `/안건 오늘`
  - 설명: 오늘 안건 링크 조회
  - 권한: 전체

- `/안건 최근 [일수]`
  - 설명: 최근 N일 안건 링크 조회 (기본 7일)
  - 권한: 전체

## 회의
- `/회의 시작 [채널]`
  - 설명: 회의 시작 메시지 + 스레드 생성
  - 채널 미입력 시 설정 채널(없으면 현재 채널) 사용
  - 스레드 템플릿 액션 패널: `결정 추가`, `액션 추가`, `할 일 추가`, `회의 종료`
  - 권한: 운영진 또는 회의열기역할(초기에는 Manage Server/Administrator)

- `/회의 종료 회의아이디:<세션ID>`
  - 설명: 지정한 회의를 요약 생성 후 종료
  - 요약 액션: `요약 재생성`, `결정 추가`, `액션 추가`, `할 일 추가`, `과제로 전환`, `원문 보기`
  - 권한: 운영진

- `/회의 기록 유형:<결정|액션|투두> 내용:<텍스트> [담당자] [기한] [회의아이디]`
  - 설명: 진행 중 회의의 구조화 항목 기록
  - 스레드 밖에서는 `회의아이디` 필수
  - 권한: 전체

- `/회의 항목 동작:<조회|취소> [아이디] [회의아이디]`
  - 설명: 구조화 항목 조회/취소
  - `취소`는 `아이디` 필수
  - 스레드 밖에서는 `회의아이디` 필수
  - 권한: 전체

## 과제
- `/과제 등록`
  - 기본 동작: V2 빠른 등록(모달 + 2단계 ephemeral 선택)
  - 권한: 운영진(미설정 초기에는 Manage Server/Administrator)

### 과제 빠른 등록 V2 (기본)
- 1단계 모달(3필드)
  - `제목`(필수)
  - `마감일`(선택, 미입력 시 내일 23:59 KST)
  - `링크`(선택)
- 2단계(ephemeral)
  - 채널 선택, 역할 선택, 멘션 여부, 알림 프리셋(기본값/24h/3h/1h/사용자설정)
  - 기본값 자동 채움
    - 길드 기본값: `default_task_channel_id`, `default_notify_role_id`
    - 사용자 최근값: `guild_user_task_preferences`
  - 미리보기 + `[등록 확정] [고급옵션] [취소]`
- 알림 시각/임박 알림은 기본값으로 자동 설정

- `/과제 목록 [상태]`
  - 설명: 과제 목록 조회 (`대기/완료/종료` 또는 `PENDING/DONE/CLOSED`)
  - 권한: 전체

- `/과제 상세 아이디:<id>`
  - 설명: 과제 상세 조회
  - 권한: 전체

- `/과제 완료 아이디:<id>`
  - 설명: 과제 완료 처리
  - 권한: 운영진

## 모각코
- `/모각코 오늘`
  - 설명: 오늘 누적시간/출석 기준/1시간 목표 진행률 조회
  - 권한: 전체

- `/모각코 내정보 기간:<day|week|month>`
  - 설명: 내 누적시간/참여일/참여율 조회
  - 권한: 전체

- `/모각코 랭킹 기간:<day|week|month> [인원]`
  - 설명: 기간별 누적시간 랭킹 조회
  - 권한: 전체

- 모각코 집계 음성채널 설정
  - 추가: `/설정 마법사 모각코음성추가:#모각코-1`
  - 해제: `/설정 마법사 모각코음성해제:#모각코-1`

## 설정
- `/설정 마법사 [운영진역할] [회의열기역할] [홈채널] [회의채널] [모각코채널] [모각코음성추가] [모각코음성해제] [과제공지채널] [과제알림역할]`
  - 설명: 핵심 설정을 한 번에 적용
  - 예시: `/설정 마법사 운영진역할:@운영진 홈채널:#홈 회의채널:#회의 모각코채널:#모각코 모각코음성추가:#모각코-1 과제공지채널:#과제 과제알림역할:@알림역할`
  - 권한: 운영진 또는 Manage Server/Administrator

- `/설정 상태`
  - 설명: 현재 길드 설정 상태 조회
  - 권한: 전체

- 마법사 단계
  - 1. 권한 설정: 운영진 역할, 회의 시작 역할
  - 2. 채널 설정: 홈/회의/모각코 공지/과제 공지 채널, 모각코 집계 음성채널
  - 3. 과제 기본값: 과제 기본 알림 역할
  - 4. 상태 확인: `/설정 상태`

## 회의 음성요약
- `VOICE_SUMMARY_ENABLED=false` 기본값에서는 `/회의음성` 명령을 등록하지 않음
- enabled=true일 때도 현재는 skeleton(상태 전이) 중심

## 레거시 호환 (숨김/점진 제거 대상)
- 회의: `/결정`, `/액션`, `/투두`, `/회의 항목조회`, `/회의 항목취소`
- 안건: `/안건 생성`
- 과제: `/과제 삭제` (취소 상태 전환)
- 모각코 채널 설정: `/모각코 채널 등록|해제|목록`
- 설정 세부 명령: `/설정 운영진역할`, `/설정 회의채널`, `/설정 과제알림역할` 등

## 운영 메모
- 명령어 이름은 `모각코`로 고정 (`모각고` 오타 금지)
- 데이터 기준 시간대는 `Asia/Seoul`
- 상세 정책(권한/에러/스키마)은 `DOCS/API_SPECIFICATION.md`를 기준으로 유지
